Una **reimplementación multiplataforma** impulsada por la comunidad de Plants vs. Zombies: Game of the Year Edition, con el objetivo de ofrecer la **experiencia 100% auténtica** de Plants vs. Zombies en cualquier plataforma.

|            🌿 Auténtico            |                                                  🎮 Portátil                                                 |     🛠️ Abierto     |
| :--------------------------------: | :----------------------------------------------------------------------------------------------------------: | :-----------------: |
| Recreación casi completa del juego | Compatible con sistemas 32/64 bit<br>Funciona en Linux, Windows, macOS, Android, iOS, WebAssembly, Switch... | OpenGL ES 2.0 & SDL |

🌐 **¿No quieres instalarlo?** [Prueba directamente en tu navegador](https://IsraelDXPP.github.io/PvZ-Portable/pvz-portable-wasm/pvz-portable.html) (Aún necesitas tus propios archivos de juego).

**⚠️ Aviso:**

* Este repositorio **NO** contiene ningún recurso con copyright (imágenes, música o fuentes) de PopCap Games o Electronic Arts. Los usuarios deben proporcionar su propio `main.pak` y la carpeta `properties/` de una **copia legalmente adquirida** de Plants vs. Zombies: GOTY Edition.
* La base de código es una reimplementación manual basada en investigación de la comunidad (como [植物大战僵尸吧](https://tieba.baidu.com/f?ie=utf-8&kw=%E6%A4%8D%E7%89%A9%E5%A4%A7%E6%88%98%E5%83%B5%E5%B0%B8), [PVZ Wiki](https://wiki.pvz1.com/doku.php?id=home) y [PvZ Tools](https://pvz.tools/memory/)). Está escrita para usar backends portátiles como SDL2 y OpenGL ES 2.0 (con fallback a OpenGL 2.1 en escritorio). El autor (wszqkzqk) **NUNCA hizo ingeniería inversa** del programa; todo se basó en información pública y pruebas de juego. Además, el código generado mediante ingeniería inversa **no será aceptado**.
* Este proyecto es únicamente con fines **educativos**, enfocado en **técnicas de portabilidad multiplataforma**, modernización del motor y aprendizaje de cómo la lógica de juegos clásicos puede adaptarse a diversas arquitecturas (Nintendo Switch, 3DS, etc.).
* No comercial: Este proyecto no está afiliado, autorizado ni respaldado por PopCap Games o Electronic Arts.
* Los íconos del proyecto y logotipos de plataformas fueron creados por mí (wszqkzqk) con ayuda de herramientas de generación de imágenes por IA y no son activos oficiales de PopCap/EA.
* Para jugar usando este proyecto **DEBES** tener los archivos originales del juego comprándolos en [EA](https://www.ea.com/games/plants-vs-zombies/plants-vs-zombies) o [Steam](https://store.steampowered.com/app/3590/Plants_vs_Zombies_GOTY_Edition/).
